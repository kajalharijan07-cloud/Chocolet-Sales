# Chocolate Sales Dataset (Chocolate.xlsb)

## Overview
This repository contains the **Chocolate.xlsb** file, which is a binary Excel workbook.
The dataset is intended for practice and analysis of chocolate sales data using tools
such as Excel, Power BI, Python, or SQL (after conversion).

## File Description
- **Chocolate.xlsb**: Binary Excel file containing chocolate-related sales data.

## Possible Columns (example)
Depending on the sheet structure, the dataset may include:
- Product / Chocolate Name
- Category
- Region / Country
- Sales Amount
- Quantity Sold
- Date / Time

## How to Use
1. Open `Chocolate.xlsb` in Microsoft Excel.
2. Enable editing if prompted.
3. Review the sheets and column headers.
4. (Optional) Convert to `.xlsx` or `.csv` for use with Python or SQL.

## Tools You Can Use
- Microsoft Excel
- Power BI
- Python (pandas)
- SQL (after importing data)

## Notes
- `.xlsb` format is optimized for performance and smaller file size.
- Make sure macros are enabled only if the file is from a trusted source.
